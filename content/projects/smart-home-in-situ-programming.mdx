## Overview

This project studies smart-home programming as an iterative, in-situ process rather than a preset GUI-only workflow.

## Contribution

It introduces snapshot-based automation configuration, simulation-based testing, and design guidance for dynamic
programming flow, interface clarity, and scene management.
