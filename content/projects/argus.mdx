## Overview

ArgusSense is a multi-agent framework that automates the deployment gap in smart environments.

## Core Idea

Using VLMs and LLMs for spatial grounding, it transforms user needs into optimized sensor plans. By replacing manual
expert analysis with AI-driven reasoning, ArgusSense aims to make intelligent, context-aware spaces faster and more
cost-effective to build.
