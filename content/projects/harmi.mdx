## Overview

HARMI enhances activity recognition with an extended temporal horizon and stronger context understanding based on LLMs.

## Core Idea

The framework progressively captures real-time activity features and structures their trajectories into long-term memory,
so the model can correct recognition results using a global historical perspective.
