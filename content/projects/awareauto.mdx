## Overview

AwareAuto introduces LLMs into automation generation for end-user programming in smart environments.

## Core Idea

The system supports contextual, multimodal, and flexible user expression for dynamic parameters, multiple branches, and
temporal constraints that are difficult to express in traditional automation tools.
